# Question service

Serviço responsável por gerenciar as perguntas do usuário.

## Instalação

```bash
$ npm install
```

## Rodando a aplicação

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Rodando os testes

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
